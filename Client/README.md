# ITAM Web Application

IT Asset Management (ITAM) frontend for a telecom context, with a rule-based predictive maintenance module.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Project Structure

- src/components/ui: reusable UI and layout foundation components
- src/components/ComponentShowcase.tsx: minimal component demo screen
- src/utils: shared utilities
